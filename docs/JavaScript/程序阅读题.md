- 下面代码的输出是什么

```js
function Car() {
  this.make = "Lamborghini";

  return { make: "Maserati" };
}

const myCar = new Car();

console.log(myCar.make); // Maserati
```

返回属性的时候，属性的值等于 返回的 值，而不是构造函数中设定的值。

- 下面的代码输出什么(函数的 length 属性)

```js
(() => 1).length === 0; //true
```

函数是有 length 属性的，函数的 length 属性就是函数参数的个数
